# Web_318421997

Web course project

הסבר קצר על הרעיון:
מערכת להזמנת חדרי פגישות לפי שעה. ניתן להזמין חדר פגישות עם סאונד, עם מקרן, עם וויפיי, עם מיטה טיפולית וכו'...

סדר פעולות באתר:
לקוח מתעניין נכנס ויכול להסתכל על דף הבית שם הוא יכול לקרוא קצת עלינו ולקרוא המלצות. לאחר מכן הוא יכול להכנס לדף הזמנת חדר או לדף יצירת קשר. בכל מקרה בסוף מילוי כל טופס (יצירת קשר או הזמנת חדר) הלקוח יועבר לדף תודה.

דגשים לחלק ג':
לצורך ביצוע חלק זה, למדתי ועשיתי כמה דברים שהם מעבר לחומר הנלמד בכיתה ואשמח לשים את הדגש עליהם:
1. הכנסת הערכים לעמוד פאג לא נעשתה לטבלה פשוטה אלא לפורמט של בלוקים.
2. כאשר מתבצעת הזמנה, נוצר לקוח (במידה והלקוח לא קיים) ואז מעלים את מספר ההזמנות שביצע ואז מכניסים את הזמנת החדר.
3. כאשר לקוח יוצר קשר הוא קודם כל נוצר כלקוח במערכת ורק אז נוצרת פנייה שמועברת לטבלה בבסיס הנתונים.
4. לא היה חובה לממש View Engine ועשיתי את זה.
5. בניתי את המערכת ככה שכאשר חדר מסויים הוזמן בכל השעות ביום ספציפי, לא יראו אותו כאופציה להזמנה. (ניתן לבצע בדיקה על הזמנת חדר ב-28.2.2023 ולראות שחדר מספר 1 לא מופיע).
6. בגלל שבדיקות הקלט שלי נעשות בקובץ JS בצד הלקוח, למדתי איך לשלוח קריאת פוסט מצד הלקוח ב-JS ולא רק דרך הטופס. ניתן לראות שיש שתי קריאות שנעשות מצד הלקוח באמצעות קובץ ה-Main.JS ויש קריאה אחת שנעשית באופן בו נלמד בכיתה. ככה שמרתי על בדיקות תקינות הקלט.

בהצלחה!
